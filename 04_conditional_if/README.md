## Тема: Условный оператор if

### Задача 1. Бот-приветствие

**Дано:** Напиши программу, которая спрашивает имя пользователя и его возраст.

- Если имя — "Admin", выведи "Привет, босс!".
- Если возраст меньше 18, выведи "Привет, [имя]! Тебе еще рано сюда.".
- Во всех остальных случаях — "Привет, [имя]!".

**Решение:**

```python
user_name = input("Введите имя: ")
user_age = int(input("Введите возраст: "))

if user_name.lower() == "Admin":
    print("Привет, босс!")
elif user_age < 18:
    print(f"Привет, {user_name}! Тебе еще рано сюда.")
else:
    print(f"Привет, {user_name}")
```

### Задача 2. Валидатор почты

**Дано:** Напиши код, который проверяет строку на соответствие базовым правилам почты:

1. Строка должна содержать символ `@`.
2. Строка должна заканчиваться на `.com` или `.ru` (подсказка: метод `endswith()` принимает кортеж, например `.endswith(('.com', '.ru'))`).
3. Если всё ок — "Почта принята", иначе "Неверный формат".

**Решение:**

```python
email = input("Введите email: ")

if "@" in email and email.endswith((".com", ".ru")):
    print("Почта корректна")
else:
    print("Почта неверна")
```

### Задача 3. Калькулятор скидок

**Дано:** Пользователь вводит сумму покупки и категорию клиента ("VIP", "Regular", "New").

- VIP — скидка 20%.
- Regular — скидка 10%.
- New — скидка 5%.
- Выведи итоговую сумму после скидки. *Не забудь про проверку: если категория введена неверно, выведи "Неизвестная категория".*

**Решение:**

```python
sum_of_buying = int(input("Введите сумму покупки: "))
categoria_of_client = input("Введите категорию клиента: ")

if categoria_of_client.lower() == "vip":
    print("Отлично! Вам дана скидка в 20%")
    print(f"Итоговая сумма: {sum_of_buying * (1 - 0.20)}")
elif categoria_of_client.lower() == "regular":
    print("Прекрасно! Вам дана скидка в 10%")
    print(f"Итоговая сумма: {sum_of_buying * (1 - 0.10)}")
elif categoria_of_client.lower() == "new":
    print("Хорошо! Вам дана скидка в 5%")
    print(f"Итоговая сумма: {sum_of_buying * (1 - 0.05)}")
else:
    print("Неизвестная категория")
```

### Задача 4. Анализатор строки

**Дано:** Дана строка: `"  apple, banana, cherry  "`.

1. Преврати её в список фруктов (очисти от пробелов и разбей по запятой).
2. С помощью `if` проверь, есть ли в списке "banana". Если есть — выведи "Бананы на месте!", если нет — добавь их в список.
3. Выведи итоговый список обратно в строку через `|`.

**Решение:**

```python
fruits = "  apple, banana, cherry  "

new_fruits = [f.strip() for f in fruits.split(",")]

if "banana" in new_fruits:
    print("Бананы на месте!")
else:
    new_fruits.append("banana")

print(f"Итоговый список: {' | '.join(new_fruits)}")
```

### Задача 5. Система авторизации

**Дано:** У тебя есть «база данных» (список логинов): `users = ["admin", "guest", "user1"]`.
Пользователь вводит логин и пароль.

1. Сначала проверь, есть ли логин в списке (используй `in`).
2. Если есть, проверь пароль: он должен быть равен "12345".
3. Если всё верно — "Доступ открыт".
4. Если логина нет — "Пользователь не найден".
5. Если логин есть, но пароль неверный — "Неверный пароль".

**Решение:**

```python
users = ["admin", "guest", "user1"]

user_login = input("Введите логин: ")
user_password = int(input("Введите пароль: "))

if user_login in users:
    if user_password == 12345:
        print("Доступ открыт")
    else:
        print("Неверный пароль")
else:
    print("Пользователь не найден")
```

### Задача 6. Инженерный конструктор

**Дано:** Пользователь вводит список покупок через запятую: `"молоко, хлеб, сыр, колбаса"`.

1. Преврати строку в список.
2. Проверь длину списка:
    - Если элементов > 5 — выведи "Слишком много покупок, удаляю лишнее!" и удали последний элемент.
    - Если элементов < 2 — выведи "Маловато покупок, добавь что-нибудь!".
    - Если элементов от 2 до 5 — выведи "Отличный список!".
3. В конце выведи итоговый список, отсортированный по алфавиту.

**Решение:**

```python
food = "молоко, хлеб, сыр, колбаса"

food_list = [item.strip() for item in food.split(",")]

if len(food_list) > 5:
    print("Слишком много покупок, удаляю лишнее!")
    food_list.pop()
elif len(food_list) < 2:
    print("Маловато покупок, добавь что-нибудь!")
elif 2 <= len(food_list) <= 5:
    print("Отличный список!")

food_list.sort()
print(f"Итоговый список: {food_list}")
```
