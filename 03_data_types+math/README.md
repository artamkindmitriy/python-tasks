### Задача 1. **Ставки на спорт**

**Дано:** Нас наняла букмекерская контора, где проводятся ставки на конный спорт. Напишите программу расчёта потенциального выигрыша игрока. Для этого вводится его ставка в рублях и коэффициент (вещественное число), а выводится их произведение в качестве потенциального выигрыша.

*Пример:*

```
Сколько ставим? 1234
Какой коэффициент? 1.716
Потенциальный выигрыш: 2117.544
```

Усложнение задачи: сделайте так, чтобы после точки выводилось не больше двух цифр.

**Решение:**

```python
stavka = int(input("Сколько ставим? "))
coef = float(input("Какой коэффициент? "))

result = stavka * coef
print(f"Потенциальный выигрыш: {result:.2f}")
```

### Задача 2. **День рождения**

**Дано:** В честь дня рождения сына отец не придумал ничего лучше, кроме как подарить денег. Зато он придумал хоть и странную, но интересную формулу, по которой высчитывается сколько он подарит: возраст сына умножается на 1.5 и на его температуру тела. Остаётся только позавидовать такой фантазии.

Напишите программу, которая запрашивает у пользователя возраст (целое число) и температуру тела (вещественное число) и в результате выводит сколько отец подарит сыну денег на день рождения.

**Решение:**

```python
age = int(input("Сколько лет сыну? "))
temperature = float(input("Какая температура тела? "))

result = age * temperature * 1.5
print("Отец подарит сыну:", round(result, 2), "рублей")
```

### Задача 3. **Индекс массы тела**

**Дано:** Алексей работает диетологом в частной клинике, каждый день он принимает пациентов разных возрастов и с разными показателями роста (в метрах) и веса (в кг). Для каждого человека ему нужно считать индекс массы тела - это вес поделить на рост в квадрате. По государственным стандартам индекс округляется до двух знаков после точки. Затем по этому индексу определяется, всё ли в порядке у человека с массой тела: если до 18.5, то недобор; до 25 - всё нормально, до 30 уже идёт избыток, а после 30 - ожирение. Напишите такую программу для Алексея.

**Решение:**

```python
meters = float(input("Введите рост: "))
weight = float(input("Введите вес: "))

index_of_body = round(weight / (meters ** 2), 2)

if index_of_body <= 18.5:
    print("Это недобор!")
elif index_of_body <= 25:
    print("Это норма")
elif index_of_body <= 30:
    print("Это избыток!")
else:
    print("У вас ожирение!")
```

### Задача 4. **Космические рейнджеры**

**Дано:** В далеком (а может и не очень) будущем на некоторой планете можно купить космический корабль за пол-кредита (CR). Один CR это 2200 чатлов. Причем чатлы неделимы и являются всегда целым числом. Напишите простую программу-конвертор валют. В программу вводится количество чатлов, а она сообщает сколько это CR и сколько кораблей можно купить на эту сумму.

*Пример 1:*

```
Сколько чатлов? 3150
Это 1.4318181818181819 CR
Можно купить кораблей: 2
```

*Пример 2:*

```
Сколько чатлов? 4400
Это 2.0 CR
Можно купить кораблей: 4
```

**Решение:**

```python
chatls = int(input("Сколько чатлов? "))

```

### Задача 5. **Конвертация**

**Дано:** При оплате покупок картой за рубежом банки делают конвертацию через промежуточную валюту. Например, если оплачивать отечественной картой товар в евро, то сначала эта сумма конвертируется в доллары, а потом — в рубли.

Что нужно сделать

Напишите программу, которая получает на вход стоимость покупки в евро, а затем выводит ответ в рублях. Представим, что мы живём в альтернативной реальности, где 1 евро = 1.25 доллара, а 1 доллар = 60.87 рубля.

### Задача 7. Валидатор нормализации весов

**Дано:** На вход подаются два веса нейронной сети (float). Напишите программу, которая складывает их. Проверьте с помощью `math.isclose`, равна ли сумма единице (с погрешностью `1e-5`). Если равна — выведите "Сумма равна 1", иначе — "Ошибка нормализации".

**Решение:**

```python
import math

first_weight = float(input("Введите первый вес: "))
second_weight = float(input("Введите вторый вес: "))

if math.isclose(first_weight + second_weight, 1, abs_tol=1e-5):
    print("Сумма равна 1")
else:
    print("Ошибка нормализации")
```

### Задача 8. Округление тензора

### Задача 9. Расчет потерь

**Дано:** Представьте, что вы считаете функцию потерь. Напишите программу, которая запрашивает у пользователя значение вероятности `p` (от 0 до 1). Вычислите `-math.log(p)` (натуральный логарифм). Выведите результат, округлив его до 4 знаков после точки с помощью `round()`.

**Решение:**

```python
import math

p = float(input("Введите вероятность (от 0 до 1): "))

if p > 0:
    result = round(-math.log(p), 4)
    print(f"Потеря: {result}")
else:
    print("Ошибка: вероятность должна быть больше 0")
```

### Задача 10. Анализатор системных логов

**Дано:** В программу приходит строка-лог вида `"[ERROR] 2026-06-10: Connection timeout"`.
Напишите программу, которая извлекает из этой строки:

1. Тип события (всё, что внутри квадратных скобок).
2. Текст сообщения (всё, что идет после двоеточия и пробела).
3. Приведите текст сообщения к верхнему регистру.
*Подсказка: используй методы поиска и срезы.*

**Решение:**

```python
error_str = "[ERROR] 2026-06-10: Connection timeout"

start_type = error_str.find("[") + 1
end_type = error_str.find("]")

event_type = error_str[start_type:end_type]
print(f"Тип события: {event_type}")

find_index = error_str.find(":")
event_index = error_str[find_index + 2:]
print(f"Текст сообщения: {event_index}")
print(f"Текст в верхнем регистре: {error_str.upper()}")
```

### Задача 11. Маскировщик персональных данных

**Дано:** На вход подается строка с Email пользователя, например alex.python_master@innopolis.university.
Напишите программу, которая скрывает часть почты: первые 3 символа имени остаются, остальное до символа @ заменяется на три звездочки ***, а домен остается без изменений.

*Пример:*

```
ale***@innopolis.university
```

**Решение:**

```python
email = "alex.python_master@innopolis.university"

result = email[:3] + "***" + email[email.find("@"):]
print(result)
```

### Задача 12. Система доступа в лабораторию

**Дано:** У нас есть три условия для входа:

1. Наличие пропуска (`True`/`False`).
2. Наличие защитного халата (`True`/`False`).
3. Время визита (число от 0 до 24). Лаборатория открыта с 9 до 18 часов включительно.
Напишите программу, которая запрашивает эти данные и выводит `True` (доступ разрешен) только если есть пропуск И халат, И время в пределах графика.

**Решение:**

```python
time_visiting = int(input("Время визита: "))
is_pass = input("Есть пропуск (True/False)? ") == "True"
is_dressing = input("Есть защитный халат (True/False)? ") == "True"

if 9 <= time_visiting <= 24 and is_pass and is_dressing:
    print("Доступ разрешен")
else:
    print("Доступ запрещен")
```
